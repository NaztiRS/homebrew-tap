# NaztiRS Homebrew Tap

Casks for [NaztiRS](https://github.com/NaztiRS) apps.

## Downbender

[Downbender](https://github.com/NaztiRS/downbender) — download videos or
extract MP3, natively on your Mac.

```bash
brew install --cask naztirs/tap/downbender
```

Downbender is not notarized by Apple, so macOS blocks the first launch once:
**System Settings → Privacy & Security → Open Anyway** (or install with
`--no-quarantine`).
